I, Cấu trúc cố định ban đầu để bắt đầu thực hiện 1 chương trình thư viện three.js 
    import * as THREE from 'three';
    const scene = new THREE.Scene();
    const camera = new THREE.PerspectiveCamera( 75, window.innerWidth / window.innerHeight, 0.1, 1000 );
    const renderer = new THREE.WebGLRenderer();
    renderer.setSize( window.innerWidth, window.innerHeight );
    document.body.appendChild( renderer.domElement );
II, Phương thức tạo 1 khối: dùng cú pháp khai báo 1 biến =  new + <Lớp đối tượng (dữ liệu vào )>
    dưới đây là 1 số hình học có sẵn:
    | Hình dạng                      | Lớp đối tượng                                                               | Cú pháp                                                                                                                                                                                                      | Giải thích                                 |
    | ------------------------------ | ----------------------------------------------------------------------------| -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------| ------------------------------------------ |
    | Hộp / Hình chữ nhật            | THREE.BoxGeometry                                                           | new THREE.BoxGeometry(chiều rộng, chiều cao, chiều sâu)                                                                                                                                                      | Tạo khối hộp (cube hoặc rectangular box).  |
    | Cầu                            | THREE.SphereGeometry                                                        | new THREE.SphereGeometry(bán kính, số lượng kinh tuyến, số lượng vĩ tuyến)                                                                                                                                   | Tạo hình cầu (ball).                       |
    | Mặt phẳng                      | THREE.PlaneGeometry                                                         | new THREE.PlaneGeometry(Chiều rộng mặt phẳng, Chiều cao mặt phẳng, Số đoạn chia theo chiều ngang, Số đoạn chia theo chiều dọc)                                                                               | Tạo mặt phẳng 2D (flat surface).           |
    | Hình trụ                       | THREE.CylinderGeometry                                                      | new THREE.CylinderGeometry(Bán kính mặt trên, Bán kính mặt dưới, Chiều cao trụ, Số đoạn chia quanh trục)                                                                                                     | Tạo ống hình trụ (tube hoặc column).       |
    | Hình nón                       | THREE.ConeGeometry                                                          | new THREE.ConeGeometry(Bán kính đáy, Chiều cao, Số đoạn chia quanh trục)                                                                                                                                     | Tạo khối nón.                              |
    | Hình vòng tròn                 | THREE.CircleGeometry                                                        | new THREE.CircleGeometry(Bán kính, Số đoạn chia)                                                                                                                                                             | Tạo đĩa tròn phẳng.                        |
    | Hình vòng donut (xuyến)        | THREE.TorusGeometry                                                         | new THREE.TorusGeometry(bán kính vòng lớn, bán kính ống nhỏ, số đoạn chia quanh vòng lớn, số đoạn chia quanh ống nhỏ)                                                                                        | Tạo hình donut 3D.                         |
    | Hình vòng xoắn ốc (torus knot) | THREE.TorusKnotGeometry                                                     | new THREE.TorusKnotGeometry(bán kính vòng lớn, bán kính ống nhỏ, số đoạn chia quanh ống nhỏ, số đoạn chia quanh vòng lớn, số vòng xoắn quanh trục chính, số vòng xoắn quanh ống)                             | Giống hình donut nhưng xoắn kiểu nút thắt. |
    | Hình bát cầu (phần cầu)        | THREE.SphereGeometry + phiStart, phiLength, thetaStart, thetaLength         | Ví dụ: new THREE.SphereGeometry(1,32,16,0,Math.PI)                                                                                                                                                           | Tạo một phần của hình cầu.                 |
